## Design

- Cadence: monthly (off-cadence: a major model release)
- Subjects: models in the foundation-models catalog, at most 3 efforts per model (lowest, intermediate, highest)
- Metrics: ttftMs (ms, lower-is-better), throughputTokensPerSec (tokens/s, higher-is-better), totalLatencyMs (ms, lower-is-better)
- Trials per run: 3 — instrument v2: the unified speed probe repeats 3 times per configuration for stdDev; structural probes run once
- Cost per run: ≤ $60 — shared `npm run compare` sweep (instrument v2, 47 configs); estimated ~$10 cold / ~$6 warm-started; run --estimate before each real run
