# research-reports

Finished research result pages, one Markdown file per research
(`<slug>.md`). Each page carries `description` (and optional `title`) frontmatter
and an objective, verifiable write-up that documents the exact commands,
credentials, and expected cost to reproduce the research.

These pages are previewed by the VitePress site and published to the corporate
site's `/research` section by `scripts/publish-research.sh`.
