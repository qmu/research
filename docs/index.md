# qmu research

Foundational research you can reproduce. Each study ships as runnable code in
this repository; the pages here are the written-up results.

## Browse

- [LLM基礎検証](/llm-foundation/) — canonical Japanese articles for
  foundation-model and RAG backend comparisons.
- [Research reports (source)](/research-reports/) — English reports, data
  artifacts, and history kept as reproducible source material.
- [Glossary](/glossary) — the project's shared vocabulary.
- [Dependency decisions](/dependency-decisions) — why each external dependency
  was adopted.

## Reproduce a study

Every report documents the exact commands, required credentials, and expected
cost to reproduce it. Clone the repository, install with `make install`, and run
the study's command as listed on its page.
