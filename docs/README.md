# docs

Documentation for the research monorepo.

- `research-reports/` — finished research result pages (Markdown). Previewed by
  the VitePress site and published to the corporate site.
- `adr/` — architecture decision records.
- `research-development-guideline.md` — the proposal-first protocol and the
  snapshot/history article structure.
- `glossary.md` — the project glossary (one concept, one word).
- `dependency-decisions.md` — the dependency-decision log.

The VitePress preview site is rooted here (`.vitepress/`, added by ticket
`20260622191215`).
