# Multi-model LLM comparison engine etc

## Summary

Initializes the public research monorepo and builds a living, reproducible LLM research platform: a 59-configuration multi-provider comparison, a 4-backend RAG benchmark, six published topics with Japanese reports, and a v2 measurement instrument cutting a recurring real run from about $46 to about $3.

## Key Changes

- Multi-model LLM comparison engine: adaptive probes, judge reviews, and recurring incremental sweeps with committed history across Anthropic/OpenAI/Google/xAI
- RAG benchmark measuring retrieval quality and operational cost across sqlite-vec, OpenAI File Search, AWS S3 Vectors, and Cloudflare Vectorize/AutoRAG
- Publishing boundary reversal: canonical reports live in this repository with enforced 7-section outlines; Japanese pages and indexes generate from shared metadata
- Living research structure: proposal-first Research Development Guideline (ADR 0005) with snapshot articles over dated trial history, llm-speed migrated as reference
- Instrument v2: unified speed probe, warm-started schema search, batched information accuracy — a recurring real run now costs about $3
